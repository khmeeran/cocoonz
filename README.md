# 🦋 Cocoonz — Premium Early Childhood Development Centre

## Project Overview

This is a **complete rebuild** of the Cocoonz preschool website — transformed from a basic preschool site into an **ultra-premium, corporate-level** brand website designed to feel like a **top-tier international preschool**.

---

## 🚀 How to Use

### Option 1: Direct File Open
Simply double-click `index.html` to open in any browser.

### Option 2: Local Server (Recommended)
```bash
cd website/
python -m http.server 8080
```
Then visit: `http://localhost:8080`

### Option 3: Deploy
Upload the entire `website/` folder contents to any web host:
- Netlify (drag & drop)
- Vercel
- GitHub Pages
- Any static hosting
